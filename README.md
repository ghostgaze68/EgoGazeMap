# EgoGazeMap

This repository contains the implementation of our gaze mapping pipeline, a tool designed to accurately map (egocentric) mobile eye tracking data.
*We will make the code publicly available upon acceptance. In the meantime, it provides demo videos of the mapped gaze data onto a reference image after applying the gaze mapping pipeline.
